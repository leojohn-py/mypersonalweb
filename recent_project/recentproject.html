1. Research Project

This is my MSc research paper with a title "Object Detection and Localization on Map using Multi-Camera and Lidar PointCloud". Basically, it uses in hardware(multiple camera and 3D Lidar-Light Detection and Ranging), for software it uses(Robot operating system, MATLAB, and Python Programming). For overview how it works camera produce a real-time image which is video and lidar produce 3D world using point clouds. Camera images undergo deep learning process know as object detection, whenever imagery detected humans or cars and other stuff it create a bounding box as label and the lidar allocate the distance of that object and localize to the map because lidar throws light and bounce back to the object which calculate the distance and both device are working are working together. And gain two(2) consecutive research award.
